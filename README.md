使用方法：
1.运行搜索主文件，输入库的路径
2.运行视频场景截图
3.在SD-Trainer使用wd1.4对库的所有文件打标签
4.运行合并插入标签

分割场景和插入标签都有记录，不会重复对已运行过的文件重复运行
SD-Trainer也可以使用跳过已有标签的文件
视频场景截图内可设置PySceneDetect参数，具体参数参考官方文档
合并插入标签里可以选择屏蔽某些标签
Tags-zh.csv可以修改翻译
